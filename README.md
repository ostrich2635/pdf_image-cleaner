# 🖼️ PDF Image Watermark Remover

A fast, lightweight, and completely free web-based utility designed to permanently strip unwanted image watermarks from your PDF files.

## ✨ Features
* **Bulk Processing:** Upload and clean multiple PDF files simultaneously, including ZIP archives.
* **Smart Detection:** Automatically extracts and groups duplicate images/watermarks by frequency.
* **True Redaction:** This app doesn't just put a white box over the watermark—it permanently strips the drawing operators (`/Name Do`) and orphaned objects from the PDF data layer.
* **Privacy Focused:** Your files are processed entirely in memory.
